# Stick-Hexapod

廢材四足機器人

![廢材四足機器人](picture2.jpg)

用以下材料修修補補成廢材 …

- 木材部份：飛機木、冰棒棍、竹筷
- 工具部份：斜口鉗、美工刀、剪刀、保麗龍膠
- 電子電路部份
  - Arduino Uno x 1
  - HC-SR04 超音波感應器 x 1
  - 3.7V 14500可充電鋰電池（三號電池大小） x 2
  - 原型擴充板 x 1（如果你要直接把線路焊在一起就不用）
  - 一些電線
